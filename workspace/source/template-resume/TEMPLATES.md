# Resume Templates

> **No templates are installed yet.** This handoff ships without resume
> templates. Add at least one before tailoring.

## How to add one

1. Put a base resume `.docx` in this folder (`source/template-resume/`), e.g.
   `base-resume.docx`. This is your master/longest resume — tailoring copies it
   and trims/reorders per job; it never edits your original.
2. Add a row to the table below describing each template so the `/tailor` skill
   knows which baseline to start from.

## Rules the agent follows

- Tailoring starts from one of these `.docx` files — never from scratch.
- Keep the result to one page; tighten spacing/margins before cutting content.
- Truthful alignment only: never invent employers, titles, dates, metrics,
  certifications, or tools. Pull only from `candidate-considerations.md`.
